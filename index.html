Aplicacion que se conecta a MongoDB Atlas ~ CRUD



Este ejemplo permite acceder a la herramienta MongoDB Atlas y realizar ya sea Consultas, Inserciones, Actulizaciones, Borrado y Uniones (Join) a un cluster de Bases de Datos alojadas en la Nube. Servicio que proporciona AWS(Amazon Web Services) y su integracion con MongoDB.

La aplicacion permite realizar las siguientes acciones.

  • Crear BD y conexion a la Base de Datos.

    • // Configuracion para conectar con MongoDB Atlas en Red.
    var mongoose = require('mongoose');
    var MongoClient = require('mongodb').MongoClient;
    var url = "mongodb+srv://"Usuario:"Clave"@locallibrary-syvnh.mongodb.net/test?retryWrites=true&w=majority";
    MongoClient.connect(url, function(err, db) {
    if (err) throw err;
    var dbo = db.db("mydb");
    dbo.createCollection("customers", function(err, res) {
    if (err) throw err;
    console.log("Collection created!");
    db.close();
    }); });
  • Insertar datos o información en las Bases de Datos.
